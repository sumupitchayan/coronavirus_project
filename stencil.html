Data Used

  • Coronavirus Patient Info - DXY.cn
  • Weather Information API
  • Air Quality Index API
  • Geolocation API

    • Data Source

    Our primary data was sourced from an existing coronavirus patient dataset aggregated on DXY.cn, a Chinese social website for physicians, health care professionals, pharmacies and facilities. As data on Coronavirus cases around the world is not collated by any official source, we found that this was the most comprehensive and trusted source. All data is also aggregated from publicly available sources. The dataset is also updated daily to keep up with the latest information on the spread of the virus.

    Pertinent information - location, age, date and death - is available in 70% of the cases and are not biased to any specific regions. Therefore, it is largely representative of all reported Coronavirus cases. Given the locations and date, we used the Geolocation API call to find the latitude and longitude of the cases and subsequently used API calls to find weather information and air quality indices for each case.

    Cleaning Data

    The data contains all the information necessary to complete our proposed project. That is, we are able to evaluate coronavirus death rates based on factors such as location, weather, air quality and age. The data split however is disproportianate and potentially skewed to the locations with more cases. For example, China has more reported cases than other countries. Therefore, we would like to measuring the rate of death per each factor. We believe we have adequate cases to do our measurements. There are several missing values, but these are only small proportions of our data as mentioned in the previous section. There are no duplicates and there are no data type issues.

    Cases from the Grand Princess cruise, where there is no determined location of onset and is a major outlier, is omitted to remove sampling bias.

    Challenges

    We have not encountered challenges with the data so far. We will try to use the most updated data each time since the number of death cases are increasing day by day. Our next steps are to perform binary logistic regression and multiple regression on our data.
